        폰트
        
    
          
  1. font
    2. - 지정한 폰트가 없으면 다음 폰트를 실행할 수 있도록 여러개의 폰트 지정해주고 폰트 이름이 여러 단어로 지정되어있으면 큰따옴표 사용한다. 한글폰트는 영문과 한글 모두 써준다.
          
  2. font-family
    3. - serif는 삐침을 말한다. sans-serif는 삐침이 없다는 의미이다. 폰트 패밀리는 크게 5개, serif는 우아한, sans-serif는 깔끔하고 가독성이 높다. monospace는 타자기 서체이다. cursive와 fantasy는 장난스러우며 스타일리쉬한 느낌.
          
  3. font-size
    4. - 폰트 사이즈는 px,%,em단위,키워드로 설정등이 있다.
          
  4. font-style
    5. -normal, italic, oblique 중에서 선택한다.
          
  5. font-weight
    6. - bold나 normal중에서 선택한다.
        

            이를 축약해서 style,variant,weight,size,family 순서대로 작성해도 된다. size나 family는 반드시 필요하고 설정하지않은 값은 디폴트값으로 실행
